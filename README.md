# 개인 포트폴리오 웹사이트

이 프로젝트는 HTML, CSS, JavaScript를 사용하여 만든 간단한 개인 포트폴리오 웹사이트입니다.

## 파일 구조

```
웹사이트/
├── index.html
├── style.css
└── script.js
```

## 실행 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 상단 메뉴를 클릭하여 각 섹션으로 이동할 수 있습니다.

## 기능

- Home: 소개 섹션
- Projects: 프로젝트 카드들 (더미 데이터)
- Contact: 연락처 정보
- 반응형 디자인
- 부드러운 스크롤 네비게이션

## 사용자 정의

- `index.html`에서 텍스트와 링크를 수정하세요.
- `style.css`에서 디자인을 변경하세요.
- `script.js`에서 기능을 추가하세요.
